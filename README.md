## What it does
- Enter a recipe topic  
- Choose word count (500, 800, 1000, 1200, 1500 or 2000 words)  
- Hit "Generate Recipe"  
- See a funny cooking joke while it loads  
- Get a nicely formatted recipe blog post ready to copy-paste or publish

## Technologies I used
- Python  
- Streamlit (super easy for making web apps)  
- Google Gemini API (gemini-1.5-flash) for generating the content  
- Deployed on Streamlit Community Cloud (free & fast)
